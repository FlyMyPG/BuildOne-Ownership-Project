## How do I make my prints better?
  
### Filament Selection:
  - Use only PLA at first  
  - Recommended Brands:  
    - Solutech  
    - eSun  
    
### Filament Settings:  
  - Start with whatever the manufacturer recommends  
  - Print a Speed Tower  
  - Print a Temperature Tower  
    
### Slicer Settings:  
  - Speed / Quality:  
    - Line Width  
    - Layer Thickness  
    - Infill  
    - Wall Thickness
